# 東寧物語 (Tungning Story)

![東寧物語 Logo](public/images/logo-tungning-bg.png)

> 重返 1662 年的台灣，在明鄭時期的東寧王國中經營商號、探索歷史，並與搭載 AI 記憶的 NPC 互動。

## 📜 專案簡介 (Project Overview)

**東寧物語** 是一款結合 **RPG 角色扮演** 與 **商號經營** 的網頁遊戲。玩家將扮演一名渡海來台的開拓者，在承天府（今台南）開設商號，與歷史人物互動，並透過經營累積財富與聲望。

本專案採用現代化的 **TALL Stack** (Tailwind, Alpine, Laravel, Livewire) 開發，並使用 **PostgreSQL** 作為資料庫，為未來的 **AI NPC 對話** 與 **向量記憶搜尋 (RAG)** 奠定基礎。

## 🛠️ 技術棧 (Tech Stack)

* **Backend**: Laravel 11, PHP 8.2+
* **Frontend**: Livewire 3, Alpine.js, Tailwind CSS
* **Database**: **PostgreSQL 16+** (支援 pgvector)
* **Dev Tools**: Vite, Composer, npm

## 🚀 快速開始 (Getting Started)

### 1. 環境需求
確保您的開發環境已安裝：
* PHP >= 8.2
* Composer
* Node.js & npm
* **PostgreSQL >= 16**

### 2. 安裝專案
```bash
# 複製專案
git clone [https://github.com/garycsrsr/tungningstory.git](https://github.com/garycsrsr/tungningstory.git)
cd tungningstory

# 安裝 PHP 相依套件
composer install

# 安裝前端相依套件
npm install
```

### 3. 資料庫設定 (PostgreSQL)

請確保您的 `php.ini` 已啟用 Postgres 驅動：
```ini
extension=pgsql
extension=pdo_pgsql
```

複製環境設定檔並修改：
```bash
cp .env.example .env
```

修改 `.env` 中的資料庫設定：
```dotenv
DB_CONNECTION=pgsql
DB_HOST=127.0.0.1
DB_PORT=5432
DB_DATABASE=tungning_story
DB_USERNAME=postgres  # 您的 Postgres 帳號
DB_PASSWORD=secret    # 您的 Postgres 密碼
```

### 4. 初始化
```bash
# 產生 Application Key
php artisan key:generate

# 執行資料庫遷移與填充 (建立資料表、NPC 楊英、測試帳號)
php artisan migrate:fresh --seed

# 建置前端資源
npm run build
```

### 5. 啟動開發伺服器
```bash
# 啟動 Laravel 伺服器
php artisan serve

# (可選) 啟動前端熱更新
npm run dev
```
現在訪問 [http://127.0.0.1:8000](http://127.0.0.1:8000) 即可開始遊戲！

## 🎮 遊戲特色功能

### 🏪 商號系統 (Shop System)
* **沉浸式開店流程**：與戶部主事 **NPC 楊英** 互動，簽署官契。
* **資源管理**：開店需消耗「通寶」與「體力」，體力會隨時間自動恢復。
* **動態回饋**：使用 Livewire Polling 實作體力值即時更新，無須刷新頁面。

### 🤖 AI 準備 (AI Readiness)
* **PostgreSQL 架構**：原生支援 JSONB 與高併發讀寫。
* **向量擴充 (未來規劃)**：預計導入 `pgvector`，將 NPC 對話紀錄向量化，實現具備長期記憶的智慧 NPC。

### ⚔️ 職業系統 (Career System)

玩家以 **主職業** 為起點；**兼職系統** 在設計上等於 **二轉**——並非隨劇情直接加掛副業，而是當 **主職熟練度** 累積至 **指定百分比門檻** 後，才允許 **轉職／解鎖** 對應的進階身分（敘事上仍為同一人專精更上一層的行當）。以下為設計草案。

#### 貿易地圖與航路（規劃）

海外貿易路線依區域分為 **北洋**、**南洋**、**西洋** 等版圖區塊；個別港口或商埠包含（命名可依史地再校）：**思明州**、**東洋**、**暹羅**、**爪哇**、**長崎**、**英圭黎**、**馬尼拉**、**濠鏡** 等。洋商於各地圖採購或販售 **當地特產貨物**；部分貨種的貿易資格則與 **通事** 所核發之許可證連動（見「兼職系統（二轉）」一節）。

#### 主職業

主職業決定玩家在東寧社會中的主要身分與核心迴圈；每位玩家以 **一項主職** 為軸，透過日常玩法累積 **主職熟練度**，達標後方可進行 **二轉**（解鎖下表所列之進階兼職）。具體門檻百分比（例如 80%、100% 分段）由數值與節奏另行定案。

* **洋商**：以跨海域貿易為業。可跑 **思明州**、**東洋**、**暹羅**、**爪哇**、**長崎**、**英圭黎**、**馬尼拉**、**濠鏡** 等據點，地圖層級依 **北洋**、**南洋**、**西洋** 區分航路與風險。玩法上在各地採購或拋售 **當地特產**，形成物價差與季節／事件波動。部分高價或管制貨品需持有對應 **地圖貿易許可證**（由具 **通事** 兼職身分之玩家販售），才解鎖該品項的採購或販運權限。

* **販客**：深耕島內與岸線經濟。與 **原住民** 進行在地貿易，並可經營基礎 **加工業**，將原料與半成品透過 **道具合成** 轉為可用物資。與洋商的「跨港大宗」互補，較偏向在地供應鏈與手工藝鏈。

* **衙門捕役**：代表官府維持秩序。消耗 **體力** 執行 **巡邏**，緝拿 **宵小**，並透過案件與巡段累積 **治安**（可設計為城鎮或全域數值）。玩法上與宵小玩家形成對立與博弈，並可與任務、懸賞、衙門聲望連動。

* **造船者**：專精舟楫與艤裝。消耗各類 **材料** 建造或改裝 **船隻**；成品可作為 **洋商** 進行遠航貿易時的載具條件（載重、航速、可航區域、抗風浪等，由數值與設計定之）。亦即造船者支撐海上經濟的「硬體產能」，本身亦可透過兼職 **火藥師** 延伸軍需與工礦產線。

* **宵小**：行走法外之路。可對 **其他玩家** 嘗試 **偷竊**（轉移或奪取部分物資），設計上需搭配 **成功率、冷卻、被通緝、捕役反制、安全區** 等規則，避免單邊壓制。敘事上可與兼職 **山五商行頭** 銜接，往跨政權的走私與情報網延伸。

* **儒醫**：儒學背景之醫者。消耗 **藥材** 與 **體力**，煉製 **傷藥**、**清涼散** 等成藥，供戰鬥、探索或事件使用。除醫療供給外，亦可習得 **通事** 兼職，在語言與朝貢／民間文書往來上扮演中介，並向洋商販售貿易許可，形成第二經濟來源。

#### 兼職系統（二轉）

兼職並非與主職平行的「隨時副業」，而是 **二轉職**：玩家須先將 **主職熟練度** 練至 **預設百分比以上**，系統才開放對應的 **轉職任務或儀式**（名稱可為進階考核、師承、衙門保結等），完成後始獲得進階身分。熟練度來源可綁定主職行為次數、產出、聲望或專屬量表，避免未深耕主職即跳級取得高階產能。下列為各主職 **二轉後** 解鎖的進階行當（敘事上仍稱「兼職」或「進階專精」皆可）。

* **煉糖匠**（與 **販客** 搭配）：專精糖料加工，將原料煉為 **白糖** 或 **冰糖**，供內銷、饋贈或作為更高階合成／貿易貨品的上游。

* **屯墾兵**（與 **衙門捕役** 搭配）：以軍屯或民屯邏輯產出 **糧食**，並可 **種蚵**（蚵仔／水產相關產出），支撐城鎮糧價與沿海副食品，與捕役「護境」敘事可共用同一人物弧線。

* **火藥師**（與 **造船者** 搭配）：製作 **火藥**；用途可區分為城防、船載武備或工礦爆破等，實際數值與法規（是否 PvP、是否限副本）由後續平衡決定。

* **山五商行頭**（與 **宵小** 搭配）：在 **清朝治下中國大陸** 與東寧／海上勢力之間，從事 **走私** 與 **情報傳遞**；玩法上屬高風險、高報酬的灰色路線，與宵小本職的「奪資」可形成任務鏈或勢力聲望。

* **通事**（與 **儒醫** 搭配）：通曉多方語言與文移規制，向 **洋商** 販售各 **地圖／航路貿易許可證**，使對方解鎖 **特定物品** 的貿易權限。儒醫主職提供藥品與體力迴圈，通事兼職則把「文書與口岸」變成玩家對玩家的經濟服務，與洋商主職強綁定。

#### 設計備註

* **二轉與熟練度**：進階兼職一律綁 **主職熟練度百分比門檻**，可搭配一次性轉職任務，以拉長養成、避免速成號壟斷高階產物或許可證市場。
* **洋商 × 通事**：通事兼職經濟上依賴洋商玩家購買許可；洋商則依許可拓展可販運品項，形成玩家間分工。
* **造船者 × 洋商**：船隻為跨地圖貿易的硬體門檻或加成來源，可與材料市場、政府／稅制銜接。
* **捕役 × 宵小**：治安與偷竊為對立軸，建議以機率、時段、地圖安全區與懲罰機制避免體驗失控。
* **販客 × 原住民**：與在地合成、煉糖等加工鏈串接，支撐內需與中間貨市場。
